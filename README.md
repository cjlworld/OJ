### names (命名规则)       
``` 
(nums)                      --->     from ybt
P(nums)                     --->     from luogu
L(nums)                     --->     from Loj
poj#(nums)                  --->     from poj
bzoj#(nums)                 --->     from bzoj
hdu#(nums)                  --->     from hdu
Vijos(nums)                 --->     from vijos
UVA(nums)                   --->     from UVA
CF(nums)+{A,B,C,D,E..}      --->     from Codeforces
SP(nums)                    --->     from SPOJ
acw#(nums)                  --->     from AcWing
```
